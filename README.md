## рЯІ† How It Works

| Step | What happens |
|------|-------------|
| **Upload** | User drops a `.wav` / `.mp3` / `.flac` file |
| **Separate** | [Demucs](https://github.com/facebookresearch/demucs) splits the mix into 4 stems: vocals, drums, bass, other |
| **Analyse** | CNN scans each stem in 1-second sliding windows (0.5s hop) and produces confidence scores for 11 instruments |
| **Merge** | Raw detections are grouped, gaps вЙ§ 2s are merged, and blocks < 1.5s are filtered out |
| **Output** | Timestamped instrument list displayed in-app with a `.txt` download option |
